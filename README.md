# TextTransform API

A lightweight, AI-powered text transformation Demo API built with **Spring Boot** and **Spring AI**. Uses the **Gemini GPT model** to expand, rewrite, formalize, or summarize text via a simple REST interface.

---

## Features

- Expand, rewrite, or formalize text
- Summarize long passages into concise text
- Stateless REST API
- Structured JSON responses
- Fast, easy to integrate and deploy
